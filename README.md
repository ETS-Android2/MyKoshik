# Info about app
This application was created during the course __"Samsung IT School Ukraine"__. It's a baskets for products that helps to __save money on daily products__ (milk, eggs and bread). A user can choose products from __3 supermarket chains__ (Novus, MegaMarket and Fozzy) and then __add them to his basket__. In the store user can easily __navigate his purchases and plans__. 

# Version and devices
This project is implemented in __Android Studio 3.5.2__. This app works correctly at __API 23__ : Android 6.0 (Marshmallow) and higher. Tested on Xiaomi Mi A1, Samsung Galaxy A50, Samsung Galaxy A6 and Pixel 2XL API 29 (Virtual Device), Pixel API 28 (Virtual Device) and Nexus 5 API 28 (Virtual Device).
